The Core Engine

How the Socratic Loop works

→
STEP 01
🌍
Scenario Injection
A real-world challenge is presented — "Design a water filter for a village facing drought."
→
STEP 02
💬
Student Proposes
Student answers in free-form text. The AI parses reasoning, not just the answer.
→
STEP 03
🧩
Socratic Decomposition
Wrong? Instead of "Incorrect" — the AI breaks it into a simpler sub-question to guide them there.
STEP 04
📈
Adaptive Scaffolding
Difficulty, hints, and context adapt in real-time based on the student's confidence and history.
The Platform

Built on three pillars

🤖
Socratic AI Mentor
Replaces "right/wrong" with guided problem decomposition. Adapts challenge contexts to each student's interests — sports, space, art, gaming. Never gives the answer. Always scaffolds the path.
🌱
Holistic Growth Tracker
A multi-dimensional radar chart tracking Academic Mastery, Critical Thinking, Perseverance, Collaboration, and Creative Problem Solving — replacing a single test score with a growth vector.
🧑‍🏫
Teacher Intelligence Cockpit
Real-time class map. Concept gap analysis. Holistic support alerts — emotional, social, academic — so teachers can act as mentors, not just graders. Zero extra admin load.

Tech Stack — Phase 1 MVP
🌐 HTML5 + Vanilla JS 🎨 Glassmorphism CSS 📊 Canvas Radar Charts 💾 LocalStorage / IndexedDB 🤖 Rule-based Socratic Tree 📱 Responsive Design ♿ WCAG AA Color Contrast ⚡ Zero external dependencies
Future Stack — Phase 2+

Design Constraints (Real India)
📶
Offline-first: Full functionality with zero internet. Syncs when connected.
💻
Low-spec hardware: SVG vectors, no heavy libraries, hardware-accelerated animations only.
🔒
Privacy: Student emotional/mental data encrypted, visible only to their teacher.
♿
Accessibility: ARIA labels, high-contrast mode, keyboard navigation throughout.
