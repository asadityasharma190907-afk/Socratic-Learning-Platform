### 🚀 Flagship Project

#### ⚡ [SahAI for Shiksha — Socratic Learning & Holistic Analytics](https://github.com/asadityasharma190907-afk/interactive-learning-platform)

*An offline-first, high-fidelity browser canvas empowering low-connectivity schools with guided AI mentoring and developmental tracking.*

- 📐 **Interactive Design Canvas:** Draggable components (Filter, Rooftop, Recharge Well) that snap dynamically to drop zones with spring-back physics and active feedback.
- 🎓 **Active Socratic Mentoring:** Breaks complex student problems into simple, intuitive dialogue blocks rather than displaying discouraging "Incorrect" messages.
- 📊 **Holistic Radar Analytics:** Draws real-time multi-dimensional student growth (Grit, Curiosity, Critical Thinking) directly onto responsive canvas charts.
- 📶 **Zero-dependency Architecture:** Pure vanilla JS and modular design allowing flawless operation on constrained hardware in rural areas with zero internet requirement.
